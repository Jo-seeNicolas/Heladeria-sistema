# Sistema de Control Administrativo - Heladería y Cafetería

## 📌 Descripción
Sistema web desarrollado en PHP y MySQL para la gestión de ventas, inventario, clientes y facturación de una heladería y cafetería.

## ⚙️ Funcionalidades
- Gestión de ventas por áreas (heladería y cafetería)
- Control de inventario
- Registro de gastos
- Facturación
- Gestión de clientes
- Módulo de autenticación (login/logout)

## 🛠️ Tecnologías utilizadas
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

## 🚀 Instalación
1. Descargar o clonar el repositorio
2. Copiar la carpeta en `htdocs` (XAMPP)
3. Importar la base de datos (archivo .sql)
4. Iniciar Apache y MySQL en XAMPP
5. Abrir en el navegador:
   http://localhost/heladeria-sistema
